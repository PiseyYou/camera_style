# 🎉 增强版仪表盘使用指南

## ✨ 新功能

### 1. 中英文切换 🌐
- 点击右上角的"中文"或"English"按钮切换语言
- 语言偏好自动保存到浏览器
- 下次打开自动使用上次选择的语言
- 所有界面元素都支持双语

### 2. 镜头图片展示 📷
- 新增"镜头展示"区域,显示镜头卡片
- 每个卡片包含:
  - 品牌专属颜色背景(Canon红色、Nikon黄色、Sony蓝色)
  - 镜头类型图标(定焦📸 / 变焦🔭)
  - 品牌、型号、卡口信息
  - 焦距、光圈等关键参数
- 自动从CSV数据加载
- 响应式设计,自适应屏幕大小

### 3. 保留原有功能 ✅
- 6个功能按钮(抓取、合并、分析等)
- 实时统计数据
- 自动刷新
- 历史记录
- 报告链接

## 🚀 使用方法

### 启动服务器

```bash
./start_dashboard.sh
```

服务器会自动:
1. 启动在后台
2. 打开浏览器访问 http://localhost:5000
3. 加载增强版仪表盘

### 切换语言

1. 点击右上角的语言按钮
2. 选择"中文"或"English"
3. 整个页面立即切换语言
4. 偏好自动保存

### 查看镜头展示

1. 页面加载后自动显示前6个镜头
2. 每个卡片显示镜头的关键信息
3. 不同品牌有不同的颜色主题:
   - Canon: 红色渐变
   - Nikon: 黄色渐变
   - Sony: 蓝色渐变

### 使用功能按钮

所有按钮功能与之前相同:
- 🔄 抓取数据 - 从DPReview获取新数据
- 🔗 合并数据 - 合并所有数据源
- 💰 价格分析 - 分析价格趋势
- 🔍 镜头对比 - 生成对比报告
- 🎯 获取推荐 - 个性化推荐
- ⚡ 完整分析 - 运行所有功能

## 📊 界面说明

### 顶部区域
- 标题和副标题(支持中英文)
- 语言切换按钮(右上角)

### 功能按钮区
- 6个大按钮,鼠标悬停有动画效果

### 统计卡片区
- 显示6个关键统计数据
- 自动从CSV文件读取
- 实时更新

### 报告链接区
- 快速访问各种分析报告
- 价格分析、镜头目录、对比报告等

### 镜头展示区(新增)
- 网格布局显示镜头卡片
- 每个卡片包含图片和详细信息
- 响应式设计,自适应屏幕

## 🎨 设计特点

### 颜色方案
- 主色调:紫色渐变(#667eea → #764ba2)
- 品牌色:
  - Canon: 红色(#ef4444)
  - Nikon: 黄色(#f59e0b)
  - Sony: 蓝色(#3b82f6)

### 交互效果
- 按钮悬停:上移+阴影增强
- 卡片悬停:上移+阴影增强
- 语言切换:即时响应
- 加载动画:旋转spinner

### 响应式设计
- 自适应桌面、平板、手机
- 网格布局自动调整
- 触摸友好的按钮大小

## 🔧 技术实现

### 前端技术
- 纯HTML/CSS/JavaScript
- 无外部依赖
- localStorage保存语言偏好
- Fetch API加载数据

### 数据源
- CSV文件(merged_summary.csv 或 dpreview/summary.csv)
- 自动fallback到可用数据源
- 实时解析和显示

### 国际化(i18n)
- data-zh 和 data-en 属性
- JavaScript动态切换
- 支持扩展更多语言
