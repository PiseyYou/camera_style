## 🚀 爬取更多数据

### 为什么只有6条数据?
- 这是初始测试数据
- 每个品牌只爬取了2个镜头

### 如何获取更多数据?

#### 方法1: 安装Playwright(推荐)
```bash
# 安装playwright
pip3 install playwright
python3 -m playwright install chromium

# 爬取更多数据
PYTHONPATH=. python3 scripts/ingest/dpreview_run.py --brand Canon --limit 50
PYTHONPATH=. python3 scripts/ingest/dpreview_run.py --brand Nikon --limit 50
PYTHONPATH=. python3 scripts/ingest/dpreview_run.py --brand Sony --limit 50

# 重新生成表格
python3 << 'PYTHON'
# (使用之前的生成脚本)
PYTHON
```

#### 方法2: 使用仪表盘按钮
1. 安装playwright(同上)
2. 访问 http://localhost:5000
3. 点击"抓取数据"按钮
4. 输入品牌和数量
5. 等待完成后刷新表格页面

#### 方法3: 手动添加数据
编辑 `data/parsed/dpreview/summary.csv`,添加更多镜头数据

## 📱 使用方法

### 访问表格
1. 启动服务器: `./start_dashboard.sh`
2. 访问: http://localhost:5000
3. 点击"数据表格"链接
4. 或直接访问: http://localhost:5000/lens_data_table.html

### 切换语言
- 点击右上角"中文"或"English"
- 自动保存偏好
- 下次访问自动恢复

### 返回仪表盘
- 点击左上角"← 返回仪表盘"按钮
